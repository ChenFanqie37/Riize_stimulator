# 🎮 嫂嫂模拟器

一款基于 React + TypeScript 的韩娱恋爱模拟游戏——你将扮演偶像的秘密恋人，在粉丝、公司和狗仔的夹击下，管理每一条痕迹，守护一段不能被发现的恋情。

> 哥哥你要幸福的话，只能和我...

## 📖 项目概述

嫂嫂模拟器不是单纯的恋爱模拟器，也不是单纯的危机模拟器。核心爽点是：**你知道自己和他是真的在恋爱，但全世界只能从碎片里猜；你既享受被偏爱的证明，又必须承受"证明一旦被别人看见就会变成证据"的压力。**

每一次发帖、聊天、约会、点赞、存图、看热搜，都会成为后续剧情的伏笔。游戏由 DeepSeek AI 驱动，男友和闺蜜的回复会感知你的行为、粉圈风向和危机程度。

### 核心体验

| 体验 | 玩家感受 | 系统实现 |
|------|---------|---------|
| 背着全世界恋爱 | 他在舞台上是大家的偶像，但私下给你发"到了吗" | 公开内容与私密聊天形成反差：同一天 Naver 写回归，Kakao 写想你 |
| 背德感/占有欲 | 他属于舞台，也属于你；你不能公开，但你知道真相 | 男主在聊天中出现克制占有欲：别发那张、别让他们知道你在哪里 |
| 粉圈解码压力 | 你只是发了一个杯子，粉丝能扒出同款、坐标和时间线 | 每个公开动作生成 1-4 个线索，粉圈按敏感度发酵 |
| 公司与狗仔夹击 | 公司不是反派，它关心商业后果；狗仔不是天降，它靠线索累积 | 风险分为公司风险、粉圈风险、狗仔风险、男主事业风险 |
| 恋爱不是无代价 | 越甜越危险，越危险越刺激 | 甜蜜事件提高好感，同时降低保密度或提高怀疑值 |

### 核心玩法

- 💬 **KakaoTalk 聊天** — AI 驱动，男友会引用你的 IG 线索，闺蜜会提醒危险
- 📸 **Instagram** — 发帖/Story/点赞/删帖，每个动作都在埋雷，删帖不等于消失
- 🌐 **Weverse** — 粉丝社区按阵营发声：团粉、唯粉、毒唯、站姐、路人、澄清党
- 📰 **Naver** — 只有事件足够大才上热搜，不会小事天天热搜
- 🕵️ **Dispatch** — 狗仔线索进度条：观察→跟踪→交叉验证→预告→爆料
- 📋 **公司通知** — 商业风控系统：警告/调岗/约谈/封口/隔离
- 📅 **行程表** — 约会必须从行程缝隙里找：凌晨、练习后、海外转机
- 💕 **约会系统** — 8种约会地点，每个地点有甜蜜收益和风险代价
- 🔍 **痕迹系统** — 每个行为生成线索，粉圈/公司/狗仔在不同阶段读取
- 📊 **数值系统** — 8项公开数值 + 7项隐藏风险值
- 🎭 **19种结局** — HE/OE/BE/SE/GE/狗血/暗黑

## 🔍 痕迹系统：Instagram 不是发帖，是在埋雷

玩家每次发 IG/Story/评论/点赞，都会生成一条 **Clue（线索）**。线索不一定马上爆，但会被粉圈、公司、狗仔在不同阶段读取。

### 线索类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 📍 地点 | 发帖暴露的位置信息 | 汉江夜景 → location=HanRiver |
| 🕐 时间 | 发帖时间与男友行程重合 | 深夜发文 → time=late-night |
| 👜 物品 | 同款/情侣物品 | 同款耳机 → item=couple-item |
| 🪞 反光 | 照片背景中的反光/倒影 | 窗户反光有黑色保姆车 |
| ✏️ 文案 | 暧昧/情绪化文案 | "今天风很大" → caption=ambiguous |
| 🎨 色调 | 与男友同色调/同滤镜 | 同款暖色调滤镜 |
| 📅 行程 | 与男友行程重合 | 同日出现在同一城市 |

### 玩家行为 → 线索 → 后果示例

| 玩家行为 | 生成线索 | 粉圈反应 | 男主反应 |
|---------|---------|---------|---------|
| 发汉江夜景 Story "今天风很大" | location=HanRiver, caption=ambiguous | 站姐注意到男主同晚提到"风很大" | "这张删掉吧，不是不好看，是我不想他们猜到你在哪。" |
| 发咖啡杯，杯套有男主手写字母 | item=cup, handwriting | 粉丝扒字体像签售 To 签 | "我不是怪你，是我忘了提醒你。" |
| 点赞站姐预览图又取消 | behavior=like-unlike | "她是不是手滑？她怎么会第一时间看到？" | "手滑也要挑我最好看的那张？" |
| 自拍窗户反光有保姆车 | reflection=car | 中期被扒皮帖作为第4条证据 | "下次我给你拍，至少别让玻璃出卖你。" |

## 🌀 粉圈发酵：从"姐妹这是谁"到大规模扒皮

粉圈不会无缘无故全知，按五级发酵，只有证据叠够才升级：

| 阶段 | 触发条件 | 表现 | 玩家应对 |
|------|---------|------|---------|
| 0 无感 | 线索少、严重度低 | IG 评论正常 | 可以享受甜蜜 |
| 1 眼熟 | 同地/同物出现1-2次 | "这个地方是不是他们公司附近？" | 删帖/装傻/改主页 |
| 2 小范围讨论 | Lovestagram>35 或同款>30 | 小号发帖："时间线有点巧" | 澄清/沉默/发烟雾弹 |
| 3 扒皮长帖 | 证据≥5且时间线重合>50 | 长图列证据：同款、坐标、车窗反光 | 公司介入 |
| 4 出圈争议 | 粉圈怀疑>80 或狗仔>75 | Naver/论坛搬运，路人评价 | 否认/冷处理/分开/公开 |
| 5 实锤危机 | 照片/视频/聊天截图级证据 | 公司声明、狗仔新闻、脱粉小作文 | 结局分歧 |

## 🕵️ 狗仔/Dispatch：证据拼图，不是随机炸弹

Dispatch 不会每天都出现，它像恐怖游戏里的进度条——你看不到全部，但能感受到被盯上：

| 阶段 | 前台表现 | 后台判定 |
|------|---------|---------|
| 观察 | "某上升期男团成员近期深夜行程异常" | paparazziHeat 30+ |
| 跟踪 | 约会后手机收到陌生账号关注 | paparazziHeat 50+，公开区域约会 |
| 交叉验证 | 狗仔把玩家 IG 地点、男主行程和车牌时间对上 | linkedClues ≥ 4 |
| 预告 | "某组合成员深夜同乘车辆？" | paparazziHeat 75+ |
| 爆料 | 放出低清图/剪影/时间线 | paparazziHeat 90+ |

## 🏢 公司逻辑：不是无脑反派，是商业风控

| 公司发现 | 处理方式 | 玩家感受 |
|---------|---------|---------|
| 玩家多次出现在非公开通道 | 经纪人私下提醒 | 第一次真实感受到自己不是普通恋爱对象 |
| 玩家是工作人员且私联 | 调离项目、取消通行证 | 事业线被恋爱牵连 |
| 男主回归期状态下滑 | 减少私人手机，经纪人代管 | 聊天变少，怀疑被冷处理 |
| Dispatch 预告接近实锤 | 紧急会议：否认/拖延/切断联系 | 危机章开始 |

## 💕 约会系统：甜不是目的，甜后必须留下代价

| 约会类型 | 甜蜜点 | 风险点 | 后续回声 |
|---------|--------|--------|---------|
| 练习室后门十分钟 | 他把外套披给你 | 门口CCTV、工作人员路过 | 公司通知"非工作人员请勿进入后台" |
| 深夜便利店 | 并肩吃饭团、压低帽檐笑 | 便利店监控、同款小票 | 他直播说"最近常吃那个饭团" |
| 车内短暂停留 | 雨声、低声聊天、未说出口的占有欲 | 车牌、停车场反光 | 狗仔线索+；男主占有欲+ |
| 汉江散步 | 他摘下一只耳机给你听 Demo | 路人拍背影、地标明显 | IG夜景若同步出现，Lovestagram暴涨 |
| 海外转机一小时 | 在异国机场短暂像普通情侣 | 同航班、免税店CCTV | 粉圈"她怎么也在这座城市" |

## 🧠 男主互动：爱豆男友不能只会甜言蜜语

| 状态组合 | 聊天风格 | 示例台词 |
|---------|---------|---------|
| 好感高+信任高+压力低 | 主动报备，温柔稳定 | "今天预录结束得早。你别等我到太晚，我到家给你发。" |
| 好感高+压力高 | 想靠近但容易冷处理 | "不是不想见你，是我怕我现在见到你，只会把坏情绪都给你。" |

| 事业压力爆表 | 少回复、突然道歉、深夜脆弱 | "今天台上我差点忘了走位。你不要问我怎么了。" |

## 🚀 安装指南

### 环境要求

- Node.js 18+
- npm 或 yarn

### 安装步骤

```bash
# 1. 克隆仓库
git clone https://github.com/你的用户名/riize-simulator.git
cd riize-simulator

# 2. 安装依赖
npm install

# 3. 配置 API 密钥
cp .env.example .env
# 编辑 .env 文件，填入你的 DeepSeek API Key
```

### API 密钥配置

游戏启动后也可以在设置面板中配置 API 密钥，无需编辑文件：

1. 打开游戏 → 点击底部"设置"按钮
2. 在"API 密钥"区域输入你的 DeepSeek API Key
3. 点击"+"添加
4. 点击"测试连接"验证

获取 API Key：访问 [platform.deepseek.com](https://platform.deepseek.com) 注册并创建密钥。

### 启动开发服务器

```bash
npm run dev
```

浏览器打开 `http://localhost:5173/` 即可开始游戏。

### 构建生产版本

```bash
npm run build
npm run preview
```

## 📊 数值体系

### 公开数值（8项）

| 数值 | 范围 | 作用 |
|------|------|------|
| 好感度 | 0-100 | 决定关系阶段、男主主动程度、约会甜度 |
| 信任 | 0-100 | 决定男主是否解释行程、是否愿意共同面对危机 |
| 恋情保密度 | 0-100 | 低于60出现粉圈猜测，低于30进入曝光危机 |
| 公司警觉 | 0-100 | 高于70约谈/调岗，高于85强制隔离 |
| 粉圈怀疑 | 0-100 | 高于50出现热帖，高于75出现扒皮长帖 |
| 事业压力 | 0-100 | 高时见面变少、语气变冷 |
| 生活稳定度 | 0-100 | 低时经济/学业/职场事件增多 |
| 心理承受 | 0-100 | 低时触发崩溃、删帖、逃离等事件 |

### 隐藏风险（7项，数值面板可见）

| 数值 | 作用 |
|------|------|
| 狗仔热度 | 累积到90+触发Dispatch爆料 |
| Lovestagram指数 | IG同款/同地/同日/同色调叠加 |
| 同款指数 | 饰品/手机壳/鞋/耳机等被扒 |
| 时间线重合 | 玩家和男主同地点/同时间出现次数 |
| 占有欲 | 高时聊天更有张力，但也更容易做风险行为 |
| 谣言可信度 | 粉圈扒皮的可信程度 |
| 内部泄露风险 | Close Friends泄露/工作人员泄密 |

## ❓ 常见问题

### Q: API 密钥配置后仍然无法使用？

**A:** 请检查以下几点：
1. 确认 DeepSeek 账户有余额（常见错误：HTTP 402 余额不足）
2. 在设置面板点击"测试连接"验证
3. 确认 API 地址正确（默认 `https://api.deepseek.com`）
4. 确认模型名称正确（默认 `deepseek-chat`）

### Q: 男友/闺蜜不回复消息？

**A:** AI 回复需要调用 DeepSeek API，可能原因：
1. API 密钥未配置或余额不足
2. 网络连接问题
3. API 请求频率过高（稍等片刻重试）

### Q: 如何保存游戏进度？

**A:** 点击底部"存档"按钮，支持10个存档位。存档保存在浏览器 localStorage 中，清除浏览器数据会丢失存档。

### Q: 部署到 GitHub Pages 后朋友能玩吗？

**A:** 可以！朋友打开网页后：
1. 会自动弹出 API 密钥配置引导
2. 按引导去 platform.deepseek.com 注册获取 Key
3. 在设置面板粘贴 Key 后即可开始游戏
4. 密钥仅保存在朋友自己浏览器中，安全无风险

### Q: 如何切换韩语/中文翻译？

**A:** 两种方式：
1. 聊天界面右上角的"译"按钮
2. 设置面板中的"显示韩文翻译"开关

### Q: 发帖后风险分数是怎么算的？

**A:** 痕迹引擎会根据以下因素计算：
- 公开帖基础分+20，Close Friends+5
- 包含地点关键词+15（汉江/江南/弘大等）
